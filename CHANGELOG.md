# 更新日志

## [1.1.0] - 2026-03-16

### 新增功能

- ✅ **图片消息支持**
  - 接收钉钉图片消息，自动下载到临时目录
  - 将图片文件路径转发给 OpenClaw
  - 支持 OpenClaw 回复图片，自动上传并发送到钉钉

- ✅ **文件消息支持**
  - 接收钉钉文件消息，自动下载到临时目录
  - 将文件文件路径转发给 OpenClaw
  - 支持 OpenClaw 回复文件，自动上传并发送到钉钉

- ✅ **语音消息支持**
  - 接收钉钉语音消息，自动下载
  - 转发语音文件信息给 OpenClaw

- ✅ **视频消息支持**
  - 接收钉钉视频消息，自动下载
  - 转发视频文件信息给 OpenClaw

- ✅ **临时文件管理**
  - 自动清理超过 1 小时的临时文件
  - 每小时执行一次清理任务
  - 可配置临时目录路径和 TTL

- ✅ **新增测试端点**
  - `POST /test/send-image` - 测试发送图片
  - `POST /test/send-file` - 测试发送文件

### 改进

- 升级钉钉 API 到最新版本 (`v1.0/robot`)
- 优化日志输出，增加调试信息
- 改进错误处理和异常捕获
- 增加文件上传/下载的详细日志

### 技术变更

- 添加 `form-data` 依赖用于文件上传
- 添加 `fs` 和 `path` 模块用于文件操作
- 重构消息发送逻辑，支持多种消息类型
- 增加临时目录管理功能
